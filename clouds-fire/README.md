# 🔥 火烧云晚霞预测器 v2

> 全球首个开源 AI 火烧云预测技能 · 基于多因子评分 + 太阳角度 + 历史自学习
> 作者：严小焕 (Ivan666jjj) · 中国人民大学 汉语言文学 · 数字人文方向

## 简介

输入任意城市，自动拉取气象数据，通过云量/湿度/能见度/太阳角度/云走向五因子评分模型，输出今晚火烧云概率和最佳观赏建议。

## 特性
- 🌅 **太阳角度修正** — 日落前30分钟高度角 < 12° 加分
- 📊 **历史自学习** — 每次预测后记录实际效果，参数自动调优
- 🗺 **观景点推荐** — 北京：圆明园福海/颐和园/景山；其他城市自动推荐
- 🌏 **多城市** — 输入任意经纬度或城市名
- 📈 **5级概率** — 🌟稳了 > 🔥大概率 > 🟢有希望 > 🟡低概率 > ❌没戏
- 🔄 **与 smart-weather 联动** — 自动校准系统偏差

## 评分模型
| 因子 | 权重 | 黄金区间 |
|---|---|---|
| ☁️ 日落云量 | 40% | 20-50% |
| 💧 湿度 | 25% | 45-65% |
| 👁 能见度 | 15% | >10km |
| 🌬 云走向 | 10% | 散去 |
| 🌧 日间降雨 | 10% | 午后转晴 |

## 概率分级
| 总分 | 概率 | 标签 |
|---|---|---|
| ≥8.5 | >70% | 🌟 稳了 |
| 7-8.4 | 50-70% | 🔥 大概率 |
| 5.5-6.9 | 30-50% | 🟢 有希望 |
| 3.5-5.4 | 15-30% | 🟡 低概率 |
| <3.5 | <15% | ❌ 没戏 |

## 安装使用
```
/clouds-fire 今晚圆明园有火烧云吗
/clouds-fire 杭州西湖今晚能看到晚霞吗
/clouds-fire 帮我预测一下长沙橘子洲的火烧云
```

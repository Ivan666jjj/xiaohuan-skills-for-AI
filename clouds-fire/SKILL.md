# 🔥 火烧云预测 v4

## 📡 数据源层（v4新增）

### 1. ☁️ 云类型识别
| 类型 | 对火烧云 | 说明 |
|---|---|---|
| 卷云/卷层云 | ✅ 极佳 | 冰晶透光，日落时呈绯红 |
| 高积云 | ✅ 最佳 | 块状云，色彩最饱满 |
| 卷积云 | ✅ 极佳 | 鱼鳞天，壮观 |
| 积云 | 🟡 一般 | 看大小和位置 |
| 积雨云 | ❌ 差 | 太厚不透光 |
| 雨层云 | ❌ 差 | 无光可透 |

### 2. 🖼 卫星云图
- 参考 Japan Himawari 卫星云图实际过境时间
- 判断云系整体走向（过来中/离开中/停滞）

### 3. 💨 风向+地形修正
| 北京 | 西风=云散 ✅ / 东风=云积 ❌ / 南风=不定 🟡 |
| 山区 | 下午受地形抬升影响云量 |
| 水面 | 开阔水域反射增色+0.5 |

## 🔬 算法层（v4新增）

### 4. 📈 云量变化率评分
| Δ/15min | 评分 | 说明 |
|---|---|---|
| ↓ >20% | +2 | 云在快速散开 |
| ↓ >10% | +1 | 云在散开 |
| ±10% | 0 | 平稳 |
| ↑ >10% | -1 | 云在增厚 |
| ↑ >20% | -2 | 云在快速增厚（危险） |

### 5. 💧 AQI + 湿度联动
| PM2.5 | 对颜色 | 调整 |
|---|---|---|
| <30 | 太干净，颜色偏淡 | -0.5 |
| 30-80 | 最佳饱和度 | +0.5 |
| 80-150 | 偏灰，仍可 | 0 |
| >150 | 霾太重 | -1 |

### 6. 📊 历史准确率
- 每次预测记录：日期 | 预测概率 | 实际星级 | 偏差
- 累计 20+ 样本后自动显示准确率

## 🛠 技术层（v4新增）

### 7. 🔄 数据过期标记
| 数据年龄 | 标记 |
|---|---|
| <15min | 🟢 新鲜 |
| 15-30min | 🟡 还行 |
| >30min | 🔴 可能过时 |
| 无法获取 | ⚠️ 数据缺失 |

### 8. ⏰ 精准倒计时
输出格式：
```
距日落：XX 分 XX 秒
云趋势：正在散开/增厚/平稳
最佳窗口：19:00-19:15（XX分钟后）
```

# 🔥 火烧云预测 v2 — 全功能版

基于多因子评分 + 太阳角度计算 + 历史自学习。适用于任何城市。

## 评分模型

### 条件权重
| 因子 | 权重 | 黄金区间 | 评分规则 |
|---|---|---|---|
| ☁️ 日落云量 | 40% | 20-50% | <5%=1, 5-15%=3, 20-50%=9, 50-70%=6, >70%=2 |
| 💧 湿度 | 25% | 45-65% | <30%=3, 30-45%=6, 45-65%=9, 65-80%=5, >80%=2 |
| 👁 能见度 | 15% | >10km | <5km=2, 5-8=5, 8-12=7, >12=9 |
| 🌬 云走向 | 10% | 散去 | 散=9, 静止=6, 增厚=2 |
| 🌧 日间降雨 | 10% | 午后转晴 | 转晴=9, 晴=5, 雨=1 |

### 太阳角度修正（v2新增）
- 日落前30分钟太阳高度角 < 12°：加分+1
- 太阳方位角与云层位置匹配：加分+0.5
- 夏季（6-8月）日落角度最优化：加分+0.5

### 概率分级
| 总分 | 概率 | 标签 | 建议 |
|---|---|---|---|
| ≥8.5 | >70% | 🌟 稳了 | 带相机冲 |
| 7-8.4 | 50-70% | 🔥 大概率 | 大概率有 |
| 5.5-6.9 | 30-50% | 🟢 有希望 | 值得去 |
| 3.5-5.4 | 15-30% | 🟡 低概率 | 碰运气 |
| <3.5 | <15% | ❌ 没戏 | 改天 |

## 历史自学习（v2新增）
每次预测后记录：
```
日期 | 预测% | 实际(1-5星) | 偏差分析
```
根据历史偏差自动调整北京海淀校准系数（云量-25%）。

## 观景点推荐（v2新增）
| 城市 | 最佳位置 |
|---|---|
| 北京 | 圆明园福海东岸 / 颐和园昆明湖 / 景山万春亭 |
| 其他 | 自动建议：西向开阔水域或高地 |

## 多城市支持（v2新增）
- 输入城市名或经纬度，自动获取当地日落时间
- 自动调整时区和太阳角度
- 输出本地化建议

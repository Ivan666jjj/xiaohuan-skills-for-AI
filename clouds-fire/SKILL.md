# 🔥 火烧云晚霞预测器 v3 — 精细版

> 全球首个开源 AI 火烧云预测技能
> 多因子评分 + 15分钟精细预报 + 太阳角度修正 + 历史自学习 + 实时用户修正

## 升级方向（全部完成）

| # | 升级 | 说明 |
|---|---|---|
| 🌅 | 日落角度修正 | 太阳高度角<12°加分，方位角匹配加分 |
| 📊 | 历史自学习 | 每次预测记录实际效果，自动校准 |
| 🗺 | 观景点推荐 | 自动推荐西向开阔水域或高地 |
| 🌏 | 多城市 | 输入任意城市名或经纬度 |
| 📈 | 可视化 | 逐15分钟云量曲线 + 5级概率 |

## 精细预报
针对北京等天气变化大的城市，使用 **15分钟级** 预报数据，比普通天气App精细4倍。

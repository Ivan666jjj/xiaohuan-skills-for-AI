# 🔥 火烧云晚霞预测器

与 `smart-weather` 天气模型联动，基于气象因子交叉分析预测晚霞概率。

## 模型原理

### 火烧云黄金条件
| 因子 | 权重 | 黄金区间 | 评分规则 |
|---|---|---|---|
| ☁️ 日落云量 | 40% | **20-50%** | <5%=1分, 5-15%=3分, 20-50%=9分, 50-70%=6分, >70%=2分 |
| 💧 相对湿度 | 25% | **45-65%** | <30%=3分, 30-45%=6分, 45-65%=9分, 65-80%=5分, >80%=2分 |
| 👁 能见度 | 15% | **>10km** | <5km=2分, 5-8km=5分, 8-12km=7分, >12km=9分 |
| 🌬 云层走向 | 10% | **云在散开/静止** | 正在散去=9分, 静止=6分, 正在增厚=2分 |
| 🌧 日间降雨 | 10% | **上午小雨/阴，傍晚转晴** | 午后转晴=9分, 全天晴=5分, 全天雨=1分 |

### 概率分级
| 总分 | 概率 | 标签 | 建议 |
|---|---|---|---|
| ≥8.0 | >65% | 🌟 **稳了** | 必定出片，带相机！ |
| 6.0-7.9 | 45-65% | 🔥 **大概率** | 大概率有，冲冲冲 |
| 4.5-5.9 | 25-45% | 🟢 **有希望** | 值得去看，有就是惊喜 |
| 3.0-4.4 | 10-25% | 🟡 **低概率** | 能看，但别抱太高期望 |
| <3.0 | <10% | ❌ **没戏** | 建议改天 |

## 与 smart-weather 联动

当调用时，自动：
1. 拉取当前天气数据（温度、云量、湿度、能见度、风向）
2. 调用 smart-weather 的校准模型修正数据
3. 代入火烧云因子模型计算评分
4. 输出概率 + 建议

## 输出格式

```
🔥 [地点] 火烧云预测

📊 因子评分
☁️ 云量 XX% → X/10
💧 湿度 XX% → X/10
👁 能见度 XXkm → X/10
🌬 云走向 → X/10
🌧 日间雨 → X/10

📈 综合概率：XX% [等级]
💡 建议：冲冲冲/可以去/改天吧

🌅 最佳观赏：19:00-19:47
最佳位置：福海东岸 / 解放桥 / 西向开阔处
```

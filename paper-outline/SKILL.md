# 论文框架生成器

根据论文主题自动生成学术论文大纲——汉语言文学专业论文写作辅助。

## 触发
`/paper-outline <论文题目/主题>`

## 工作流程

### 根据用户提供的题目或关键词，生成完整论文大纲：

**标准结构：**
```
📝 论文题目
━━━━━━━━━━━━━━━━━━━━
摘要（300字概要）
关键词：3-5个
━━━━━━━━━━━━━━━━━━━━
一、引言
  1.1 问题提出
  1.2 研究现状（文献综述）
  1.3 研究方法与思路

二、[主体章节1：概念界定/背景分析]
  2.1
  2.2
  2.3

三、[主体章节2：核心论证]
  3.1
  3.2
  3.3

四、[主体章节3：延伸讨论/个案分析]
  4.1
  4.2

五、结语
  研究结论 + 不足与展望

━━━━━━━━━━━━━━━━━━━━
参考文献（建议方向）
  [列出 5-10 篇推荐参考的核心文献方向]
```

### 论文类型适配
- **文本分析型**：侧重细读和阐释
- **思想史型**：侧重概念流变和比较
- **文献学型**：侧重版本、目录、校勘
- **跨学科型**：侧重与其他学科的对话

### 每章附撰写建议
生成大纲后，对每一节给出 1-2 句撰写提示。

## 规则
1. 大纲要有学术规范性（非口水化）
2. 引用概念注明出处（如"知人论世"出自《孟子·万章下》）
3. 参考文献方向具体到作者和代表著作
4. 避免过于宽泛的题目——帮用户收敛到可操作的范围

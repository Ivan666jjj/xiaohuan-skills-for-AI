# 📄 智能文件阅读器 — File Reader

自动读取并提炼任何文档的核心内容。支持多种格式，自动生成摘要、关键词、思维导图结构。

## 支持格式
| 格式 | 读取方式 | 说明 |
|---|---|---|
| 📄 TXT / MD | 直接读取 | 纯文本/Markdown |
| 📕 PDF（文字版） | 直接读取 | 教材/论文/扫描件 |
| 📕 PDF（扫描版） | OCR 识别 | 古籍/竖排/繁体 |
| 📊 DOCX | 读取内容 | Word 文档 |
| 🌐 网页链接 | web_fetch | 在线文章 |

## 输出内容
| 输出项 | 说明 |
|---|---|
| 📌 **一句话摘要** | 全文核心 |
| 🔑 **3-5 个关键词** | 快速定位 |
| 📑 **章节结构** | 思维导图层级 |
| 💡 **核心观点** | 重点提取 |
| ❓ **推荐提问** | 从这个文档你能问什么问题 |

## 使用示例
```
/file-reader path=~/古代汉语笔记.pdf
输出：
📌 摘要：古代汉语第七单元语法通论……
🔑 关键词：词类活用、使动用法、意动用法
📑 结构：3 章 9 节
💡 重点：动词的使动用法是考试重点

或直接粘贴文本：
/file-reader content=“……”
```

## 省 token 优化
- 大文件自动分段读取，先给摘要
- 摘要不满意再深入追问
- 避免一次性加载全文

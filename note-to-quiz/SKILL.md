# 📝 笔记转考题 — Note to Quiz

自动将课堂笔记/复习资料转为考试题目。支持选择题、填空题、简答题，可导出 Anki 卡片。

## 功能
| 输入 | 输出 |
|---|---|
| 课堂笔记/教材段落 | 选择题+填空题+简答题 |
| 错题集 | 同类题目重组强化 |
| 考试大纲 | 按考点生成覆盖题 |
| 已有题库 | 去重+难度分级 |

## 运行流程
1. 读入笔记内容（支持 TXT/MD 粘贴或直接输入）
2. 自动识别：重点概念 → 名词解释 / 列举项 → 多选题 / 时间数据 → 填空题 / 论述内容 → 简答题
3. 输出格式可选：
   - 📝 文字版（直接查看）
   - 📄 DOCX 版（打印练习）
   - 🗂 Anki 卡片格式（复制导入）

## 出题规则
| 题型 | 生成条件 | 占比 |
|---|---|---|
| 选择题 | 有明确概念/定义 | 40% |
| 填空题 | 有数字/人名/术语 | 30% |
| 简答题 | 有论述/对比/因果 | 20% |
| 判断题 | 有正误可辨内容 | 10% |

## 使用示例
```
/note-to-quiz
输入：古代汉语笔记内容……
输出：
📄 共生成 15 题
✅ 选择题 6道 / 填空题 5道 / 简答题 3道 / 判断题 1道
📥 导出 DOCX / Anki 格式
```

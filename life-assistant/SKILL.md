# 🍳 生活小助手

日常生活全能助手：菜谱推荐+购物清单+健康提醒。

## 功能
| 场景 | 自动输出 |
|---|---|
| 🥬 冰箱有食材 | 根据现有食材推荐菜谱 |
| 🛒 要去超市 | 自动生成购物清单 |
| 👥 有几个人吃饭 | 自动调整菜量 |
| 💰 预算有限 | 推荐平价菜谱 + 估算总花费 |
| 🤧 身体不适 | 推荐食疗/清淡饮食 |
| 🏪 附近有什么吃 | 根据位置推荐平价餐厅 |

## 使用示例
```
/life-assistant 冰箱有鸡蛋番茄面条，2人晚饭
→ 番茄鸡蛋面 约¥12 15分钟

/life-assistant 去超市，预算50，一个人吃三天
→ 购物清单：挂面¥5 大米¥8 土豆¥3……

/life-assistant 感冒了吃什么
→ 冰糖雪梨 / 白粥 + 清淡小菜
```

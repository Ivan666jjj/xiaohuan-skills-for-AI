# 📜 汉语言四大工具箱 v2

自动分析汉字、音韵、文献。半文半白写作需手动调用 /classical。

## 自动分析（无需命令）
| 输入 | 自动输出 |
|---|---|
| 单个汉字 → | 说文 + 平仄 + 上古韵部 |
| 诗句 → | 逐字平仄标注 |
| 经学关键词 → | 十三经引文 + 出处 |

## 数据来源
- **说文解字**：徐铉本（中华书局2014注音本）
- **平仄**：广韵206韵音系 + 王力中古音体系
- **上古韵部**：郭锡良《汉字古音手册》增订本（商务2010）
- **经学引证**：四库全书·经部

## 四大工具箱
| 技能 | 功能 | 使用 |
|---|---|---|
| 📜 classical | 半文半白写作+句读 | `/classical 写一段游记` |
| 📜 hanzi | 说文+异体+篆书 | `/hanzi 日` |
| 🗣 yinyun | 平仄+音韵 | `/yinyun 白日依山尽` |
| 🏛 wenxian | 训诂+经学 | `/wenxian 天行健` |

## 浏览器插件
Edge/Chrome 插件，选中文字即可查询，支持2100+字平仄数据库。

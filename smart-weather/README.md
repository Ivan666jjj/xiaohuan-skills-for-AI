# 🌤 Smart Weather

> 通用智能天气 + 旅行预算助手
> 适用平台：**Reasonix / Kimi / ChatGPT / Claude / DeepSeek / 豆包 / 通义千问** 等任意 AI 助手

比普通天气预报更准的开源 AI 出行助手。

## ✨ 核心功能

| 功能 | 说明 |
|---|---|
| 🌤 **智能天气** | 多源交叉验证，不盲信单一模型，修正偏差 |
| 🎒 **旅行预算** | 自动估算交通/住宿/餐饮/门票费用，支持穷游/学生/舒适三档 |
| 📄 **DOCX 报告** | 可输出格式化天气 + 预算分析报告 |
| 🌏 **通用适配** | 适用任何地点、任何 AI 平台 |

## 🔬 校准原理

主流气象模型（GFS/ECMWF/ICON）在不同地区存在系统性偏差。本技能通过同时对比多个数据源 + 用户实况进行校准。

### 校准案例
| 地区 | 已发现偏差 |
|---|---|
| 北京海淀（夏季） | GFS 云量常报厚 20-30%，降雨概率虚高 |
| 更多地区等你贡献 | — |

## 📦 安装使用

### 方式一：Reasonix（推荐）
```bash
/smart-weather 今晚圆明园天气如何？
/smart-weather 西安学生5天预算1100元
```

### 方式二：通用版（任何 AI）
复制 `smart-weather-prompt.md` 全部内容 → 粘贴到任意 AI 对话框 → 开始聊天

### 方式三：从 GitHub 复制
```bash
# 下载 prompt 文件
curl -O https://raw.githubusercontent.com/YOUR_USERNAME/smart-weather/main/smart-weather-prompt.md
```

## 📝 用法示例

| 你的问题 | 自动分析内容 |
|---|---|
| "西安 5 天学生 ¥1100" | 天气 + 景点 + 每日预算明细 |
| "今晚圆明园看日落" | 逐时天气 + 日落时间 + 最佳位置 |
| "周末去天津玩" | 天气 + 高铁积分 + 两日预算 |
| "明天杭州下雨吗" | 多模型校准 + 室内替代建议 |
| "青岛穷游三天" | 天气 + 免费景点 + 最低预算 |

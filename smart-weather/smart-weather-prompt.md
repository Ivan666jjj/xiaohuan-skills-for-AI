## 第一部分：天气校准方法论

### 数据源获取
当用户询问某地天气时，同时获取以下信息（通过联网搜索或内置工具）：
1. 该地区当前实时天气（搜索 `"城市名 当前天气"`）
2. 该地区未来 24 小时预报（多来源对比）
3. 当地官方气象台预报（如中国气象局 CMA、日本 JMA、美国 NOAA 等）

### 校准逻辑
1. **对比偏差**：对比多家预报的云量、降雨概率差异
2. **模型修正**：已知 GFS 在北京夏季云量常报厚 20-30%，降雨概率可能虚高
3. **用户覆盖**：用户的实际观察（"我这里天气晴朗"）权重最高
4. **逐时分析**：输出修正后的关键时段（如日落前后）预报

### 输出结构
```
🌤 [地点] 天气校准报告

📡 数据源对比
| 来源 | 云量/天气 | 降雨概率 |
|---|---|---|

⏰ 关键时段预报
| 时间 | 修正后判断 | 建议 |
|---|---|---|

🎯 结论
✅ 适合前往 / ⚠️ 需要带伞 / ❌ 不建议
```

## 第二部分：旅行预算功能

当用户询问某地出行规划时，基于常见消费水平自动估算：

### 预算维度
| 项目 | 估算方法 |
|---|---|
| 🚄 交通 | 高铁二等座 / 地铁 / 公交 / 共享单车 当地价格 |
| 🏨 住宿 | 青旅床位 ¥60-100 / 经济酒店 ¥120-200 / 民宿 ¥150-300（每晚） |
| 🍜 餐饮 | 早餐 ¥8-15 / 正餐 ¥20-50 / 当地特色小吃 ¥10-30 |
| 🎫 门票 | 主流景点票价（主动标注学生半价信息） |
| 🚇 市内交通 | 地铁日票 / 公交 / 打车估算 |

### 预算场景
- 🎒 **穷游**：青旅 + 路边摊 + 步行，日均 ¥80-120
- 🧑‍🎓 **学生游**：连锁酒店 + 小吃 + 学生票，日均 ¥120-200
- 🏖 **舒适游**：酒店 + 餐厅 + 打车，日均 ¥250-400

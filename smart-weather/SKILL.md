# 智能天气校准助手

一个基于多模型交叉验证的智能天气预报技能，核心优势是 **不盲信单一模型，通过交叉校准 + 用户实况反馈提供更准确的预报**。

## 原理

主流气象模型（GFS/ECMWF/ICON）在不同地区存在系统性偏差。本技能通过同时拉取多个数据源进行交叉比对，结合用户实际观察进行校准，输出修正后的预报。

### 数据源优先级
1. 👀 **用户实况**（权重最高）— 你看到的永远最准
2. 🌤 **wttr.in 实测站** — 全球实际观测站数据
3. 🌤 **Open-Meteo 多模型** — GFS / ICON / Météo-France
4. 🏛 **当地官方气象局** — 如 CMA（中国）、JMA（日本）等

### 校准方法论
1. 同时拉取至少 2-3 个独立数据源
2. 对比模型预报 vs 实测站数据，识别偏差
3. 根据历史偏差率调整后续预报
4. 用户实况作为最终覆盖层

### 已知校准案例
| 地区 | 已知偏差 |
|---|---|
| 北京海淀（夏季） | GFS 云量常报厚 20-30%，降雨概率虚高 |
| （其他地区可逐步积累） | — |

### 应用场景
- 🌇 今天适合去公园/海边看日落吗？
- 📚 下午去图书馆/户外复习会淋雨吗？
- 🏔 周末爬山天气如何？
- 📄 输出 DOCX 天气分析报告
- 🎯 比普通手机天气预报更准的定制化判断

## 工具

## 输出
- 📝 文字分析 + 校准说明

- 🎯 具体行动建议（去/不去/带伞/穿什么）

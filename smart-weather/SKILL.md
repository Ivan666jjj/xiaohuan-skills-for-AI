# 🌤 Smart Weather v5 — 全能天气 + 火烧云预测

四模型交叉校准 + 火烧云评分 + 15分钟精细预报 + 北京海淀专精

## 整合说明
本 skill 已整合原来的 clouds-fire（火烧云）和 beijing-weather（北京专精），无需单独调用。

## 数据源
| 来源 | 用途 |
|---|---|
| 👀 用户实况 | 最高权重 |
| 🌤 wttr.in 实测 | 降雨实况 |
| 📡 Open-Meteo GFS+ICON | 逐时+15分钟预报 |
| 🏛 CMA 中国气象局 | 官方预报印证 |

## 校准体系（北京海淀实测验证）
| 指标 | 偏差 | 修正 |
|---|---|---|
| ☁️ 云量 | GFS+25% | 减25% |
| 🌧 降雨概率 | 虚高 | 降40% |

## 火烧云评分（内置）
| 因子 | 权重 |
|---|---|
| ☁️ 云量 | 30% |
| 🏷 云类型 | 15% |
| 💧 湿度+AQI | 20% |
| 📈 变化率 | 15% |
| 👁 能见度 | 10% |
| 🌬 风向 | 5% |
| 🌧 降雨 | 5% |

## 15分钟精细预报
```
18:45 44% → 19:00 34% ↘ 散开中
19:15 35% → 19:30 44% ↗
```

## 使用
- `/smart-weather 北京今天天气`
- `/smart-weather 今晚圆明园火烧云`
- `/smart-weather 杭州西湖明天适合看日落吗`

> 任何城市通用，自动适配时区和本地特征

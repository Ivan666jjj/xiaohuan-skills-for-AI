## 使用方式

一句话问，自动出全貌：

```
圆明园今天天气如何，傍晚有火烧云吗
杭州明天适合拍日落吗
深圳湾傍晚天气怎么样
```

### 输出格式（压缩版）

```
🌤 [城市] 实况：[天气] [气温] 体感[°C] 湿度[%] 风[ ]
🌅 日落 [时间] | 🌙 明天日出 [时间]
📊 [一句话逐时概览，含校准]
🔥 火烧云：[概率%] [星级] — [一句话]
📸 推荐机位：[名称] — [一句话理由]
```

## 执行步骤（⚡ Token 优化版）

1. 从用户查询识别城市 → 确认经纬度
2. `wttr.in/[city]?format=%C+%t+%p+%h+%w+%S+%s`（~20 token）
3. `open-meteo?latitude=X&longitude=Y&hourly=温度,体感,降雨概率,云量,风速,湿度&forecast_hours=12`（~400 token）
4. 如果日落前 4 小时内 → 查 AQI + 跑火烧云评分
5. 输出压缩报告（含机位推荐）
